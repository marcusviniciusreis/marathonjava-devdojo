# Estudos de Java - Curso DevDojo
Este repositório reúne meus estudos enquanto realizo a **Maratona Java Virado no Jiraya**, curso de Java disponibilizado gratuitamente pelo canal DevDojo.
A proposta deste repositório é registrar minha evolução durante o aprendizado da linguagem Java. Aqui estão exemplos de código, exercícios e testes feitos ao longo das aulas, conforme avanço no curso.
Além de servir como prática, este espaço também funciona como um histórico do meu processo de aprendizado.

# Sobre o curso
A Maratona Java Virado no Jiraya é um curso completo que aborda a linguagem Java desde os fundamentos até conceitos mais avançados.
Durante as aulas são apresentados diversos tópicos importantes para quem deseja desenvolver aplicações utilizando Java, com foco em compreender bem os conceitos e aplicá-los na prática.

# Conteúdos estudados
Até o momento, estudei os seguinte conteudos:
* Funcionamento do Java
* Configuração do ambiente
* Tipos primitivos
* Operadores
* Estruturas condicionais
* Estruturas de repetição
* Arrays

Novos conteúdos serão adicionados ao repositório conforme avanço nas aulas.

# Tecnologias utilizadas
* Java (usando a versao 25)
* IntelliJ IDEA
